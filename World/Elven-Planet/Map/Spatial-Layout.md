# Elven Planet — Spatial Layout

See also: `Map/Plan_of_the_planet.png` for the hand-drawn reference.

---

## Overview

Hub-and-spoke design centred on [[Caer Bile]] (the world tree). Progression mixes vertical and horizontal traversal. Several one-way shortcuts flow back to the hub. This is a Soulslike/Metroidvania hybrid layout.

## Flow

1. Player arrives in the **Verdant Expanse** (intro). Linear path leads to **Caer Bile**.
2. From Caer Bile, three main routes open: **The Grove** (horizontal), **Knotroot Canopy** (horizontal then branching), and eventually **Elderwood Highlands** (via the Heartbloom Abattoir route).
3. **Knotroot Canopy** splits: descend to **Faeblight Marsh** or ascend to the spider miniboss.
4. **Faeblight Marsh** has a one-way shortcut back to Caer Bile.
5. **The Grove** connects to the **Heartbloom Abattoir** via catacombs below the wildling ziggurat.
6. **Heartbloom Abattoir** ascends to the **Elderwood Highlands** (crown of the tree).
7. **Elderwood Highlands** has a one-way shortcut back to Caer Bile after obtaining the ship material.
8. The **hidden section of the Verdant Expanse** (Alliance lab, miniboss) is accessible only by descending from the Grove.

## One-Way Shortcuts

- Faeblight Marsh → Caer Bile
- Elderwood Highlands → Caer Bile
- The Grove → Verdant Expanse hidden section (descent only)

## Cross-Planet Gate

The Faeblight Marsh lake bottom requires an item from another planet (silent water movement). First Metroidvania gate in the game.
